说明

数据来自 OpenStreetMap / Nominatim（免费、无需 Key）。英文地址为其罗马化结果，语序为“门牌+街道+区+市+省+国家”。
“按 IP 随机附近地址”：按当前出口 IP 定位所在城市，在附近随机取点并反查，只保留含街道/门牌的具体地址；结果附「按坐标验证」链接（query=纬度,经度），谷歌地图可精确落点，不受中文地址文本匹配不到的影响。
中国境内的门牌级与邮编数据覆盖不如高德/百度，罗马化的地址文本在谷歌地图按文字搜索时可能查不到——此时请用坐标链接验证（始终可定位）。
若长时间无响应，可能是网络无法访问 openstreetmap.org 或 IP 定位服务；请确认网络后重试。
