# 数据来源与资源获取指南

## 1. 本地开源 vs 线上完整版

| 项目 | 说明 |
|------|------|
| 开源仓库 | [hotbob011/real-random-taxfree-address](https://github.com/hotbob011/real-random-taxfree-address) — 引擎代码 + 预览数据，**不含**完整 JSON 资产与页面模板 |
| 完整线上 | [mockaddress.com](https://mockaddress.com/) — 9 国/地区地址 + 免税州 + 身份/信用卡/MAC |

本仓库已补齐：`src/js/main.js`、`usa-address/` `hk-address/` `uk-address/` 可运行页，以及 US/HK/UK 核心数据。

## 2. 线上 JSON 可以直接拉吗？

可以。`https://mockaddress.com/data/*.json` 为公开静态资源（HTTP 200）。

在项目根目录执行：

```powershell
powershell -ExecutionPolicy Bypass -File .\download-resources.ps1
```

或：

```bash
bash download-resources.sh
```

### 已确认的文件名

- `usData.json` / `namesData.json` / `usRealAddresses.json` / `us_taxfree.min.json`

- `hkRegions` → `data/hkData.json`
- `ukRegions` → `data/ukData.json`

## 3. 网上地址生成器的数据一般从哪来？

常见来源（按国家）：

**美国**

- [US Census Bureau](https://www.census.gov/)：州/县/城市地理与人口
- [USPS ZIP Code](https://www.usps.com/)：邮编区划（商用需授权）
- [OpenAddresses](https://openaddresses.io/)：众包街道门牌（本站 `us_taxfree.min.json` 的 `source` 字段写明 `OpenAddresses (local MG) sampled`）
- [GeoNames](https://www.geonames.org/)：地名库

**英国**

- [Royal Mail PAF](https://www.poweredbypaf.com/)：官方地址库（商用授权）
- [OS Open Data](https://www.ordnancesurvey.co.uk/) / Code-Point Open：邮编中心点（开放数据）
- Wikipedia / 公开行政区列表：构成国、城市名

**香港**

- 香港特区行政区划公开资料（港岛/九龙/新界 + 分区）
- 常见街道与商场/大厦名称（公开地图与政府资料）
- 香港**无官方邮编体系**，表单常用 `000000` 占位；跨境偶见 `999077`

**通用姓名**

- 各国统计局姓名频率表、公开姓名语料（注意许可协议）

## 5. 本地启动

```bash
# 在项目根目录
npx --yes serve -l 5173 .
# 浏览器打开：
# http://localhost:5173/usa-address/
# http://localhost:5173/hk-address/
# http://localhost:5173/uk-address/
```

必须用 HTTP 服务打开（不能 `file://`），否则 ES module 的 `fetch(data/*.json)` 会失败。
