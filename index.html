“按 IP 找附近住宅地址”：按当前出口 IP 定位所在城市，在附近由近及远随机取点反查，只保留含街道/门牌的具体地址，并剔除白宫、景点、政府/商业等地标；页面顶部徽章会明确标示是否为居民住宅地址（绿=是，黄=否）。结果附「按坐标验证」链接（query=纬度,经度），谷歌地图可精确落点，不受中文地址文本匹配不到的影响。
状态栏会显示定位到的“国家 · 城市”与数据来源，便于确认是否走了代理/VPN 的出口 IP。
数据来自 OpenStreetMap / Nominatim（免费、无需 Key）。中国境内的门牌级与邮编数据覆盖不如高德/百度，罗马化文本在谷歌地图按文字搜索时可能查不到——此时请用坐标链接验证（始终可定位）。
若长时间无响应，可能是网络无法访问 openstreetmap.org 或 IP 定位服务；请确认网络后重试。
